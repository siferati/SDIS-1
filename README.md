# SDIS Project 1 - Distributed Backup Service

## How to run

### Ubuntu

* Press ``Ctrl + Alt + T`` to open the terminal
* Navigate to ``Project`` directory
* Type ``make``
* Type ``java -cp bin Peer``

#### Javadoc

* Press ``Ctrl + Alt + T`` to open the terminal
* Navigate to ``Project`` directory
* Type ``javadoc -d docs *.java``
* Navigate to ``docs`` directory
* Open ``index.html``
